# BotForge Informational Bot (Telegram Sample)

A sample bot that demonstrates showing simple informational messages and basic navigation using BotForge.

## Run
1. Set your bot token in configuration (ApiKeys:Telegram) or pass it directly to WithTelegramBot.
2. Set news API tokens in configuration or using `dotnet user-secrets`:
- ApiKeys:WeatherAPI from https://weatherapi.com/
- ApiKeys:NewsApi from https://newsapi.org/
- ApiKeys:ExchangeRate from https://exchangerate.host/
3. Run the project.
